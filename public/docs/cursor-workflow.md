# Instalación de un servidor MCP

Primero creamos una carpeta .cursor y dentro un archivo mcp.json donde configuraremos el servidor filesystem, usando @modelcontextprotocol/server-filesystem y npx.

Después de haber guardado y reiniciado, Cursor ha sido capaz de acceder a la estructura del proyecto.

Esta herramienta puede ser util en muchas situaciones como por ejemplo:
-Analisis de código en proyectos elaborados.
-Automatiza partes muy exigentes o tardadas de la realización del proyecto.
-Obviamente la integración de la IA con diferentes herramientas lo cual agiliza mucho el trabajo.
