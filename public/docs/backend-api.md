# Axios
Es una librería de JavaScript que mejora la funcionalidad de fetch, manejando errores automáticamente, convierte JSON sin necesidad de .json() y permite configurar cabeceras globales.

# Postman
Es una aplicación para poder probar APIs REST sin necesidad de tener un frontend se usa para ver si el servidor responde con peticiones GET,POST, DELETE, etc.

# Sentry
Es una plataforma que monitoriza errores en producción, cuando ya está en producción y ocurre un error te lo notifica con todos los detalles.

# Swagger
Es una herramienta que se usa para documentar APIs REST de forma automática, haciendo una página web donde se pueden ver cada uno de los endpoints establecidos, cada parámetro que acepta y para probarlos
