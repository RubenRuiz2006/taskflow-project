# Prompts útiles 

-1 Actúa como si fueras un desarrollador senior y formula una documentación detallada de este proyecto en este archivo concreto.

    Aquí al darle un rol y una orden clara, ejecutará la tarea directamente.

-2 Desarrolla un código en lenguaje JavaScript el cual pueda hacer la información perdurable.

    Aquí especificamos pautas concretas para la realización de la tarea.

-3 Explica paso a paso este código y dime qué puedo factorizar explicándomelo.

Lo que hacemos es que no haga el trabajo y ya está, si no que explique el proceso.

-4 Actúa como si fueras profesor de programación para explicarme posibles mejoras o fallos de mi proyecto.

Aquí volvemos a darle un rol para orientar más sus respuestas a un tono educativo.

-5 Revisa mi código y cámbiame esta variable de nombre en todas las partes donde aparezca, pero déjame revisarlo antes.

Especificamos la variable que queremos cambiar y el nombre el cual le queremos dar, no sin antes revisarlo.

-6 Actualiza la paleta de colores de CSS con Colores grisáceos para un acabado más profesional.

Aquí lo que buscamos es que encuentre la mejor combinación para colores que le especificamos

-7 Compara estas dos funciones para ver cuál es más rentable y pulida.

Añadimos varias opciones posibles para que nos explique cuál es mejor y por qué

-8 Revisa posibles faltas leves que en un futuro podría generar un problema en la web, actuando como un profesor experimentado en JavaScript.

Así podemos prevenir futuros errores que serán difíciles de detectar

-9 Explica paso a paso cuál es la mejor manera de implementar esta función, con este nombre.

Explicamos la función que queremos implementar para que entienda bien la idea que tenemos para que la ejecute correctamente.

-10 Estoy realizando un proyecto sobre "...." en el cual quiero implementar una función llamada "...." con estos valores, ¿cuál es la manera más eficiente?.

Ponemos en contexto a la IA para que sea más precisa y efectiva.
