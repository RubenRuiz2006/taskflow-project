# Diferencias al programar con o sin IA

Para el primer problema le planteé a la IA que sumara todos los elementos de un array, para compararlo con la manera en que yo lo hacía:

IA: 
Tiempo-10 segundos
Resultado- 
function sumaArray(arr) {
  return arr.reduce((acumulador, elemento) => acumulador + elemento, 0);
}

Yo:
tiempo- +1 min
Resultado-
array.forEach(element => {
    let suma =+ element;
});

Podemos comprobar que la IA es mucho más eficiente y concisa, siendo una herramienta muy útil.

En mi proyecto he podido mejorar varias cosas gracias a la IA, como por ejemplo:
-Me creo mejor una función para establecer el modo oscuro, con una estructura entera que no se me había ocurrido
-Yo no había pensado que pudieran meter en los inputs espacios en blanco lo cual me ha aconsejado como cambiarlo
-Y su forma de añadir un botón de ordenar por prioridad era mucho más limpia que la mía.
